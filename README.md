# SKY GYM Ayacucho

Sitio web en React + Vite con Tailwind CSS para SKY GYM Ayacucho.

## Requisitos
- Node.js 18+
- npm 9+

## Desarrollo
- Instalar dependencias: `npm install`
- Servidor de desarrollo: `npm run dev`
- Linter: `npm run lint`
- Compilar producción: `npm run build`
- Vista previa producción: `npm run preview`

## Personalización
- WhatsApp: editar el número en `src/App.jsx` (enlace `wa.me`).
- Testimonios: editar los datos en `src/data/data.jsx`.
- Secciones: UI en `src/sections/StaticSections.jsx` y componentes en `src/components/Ui.jsx`.

## Notas
- Tailwind 4 (via `@tailwindcss/vite`) ya configurado.
- No se incluyen claves de API. Para habilitar asesoría con IA (Gemini), agrega tu API key de forma segura antes de usar.
